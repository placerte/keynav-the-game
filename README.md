# KEYNAV THE GAME
## Description

Tests your mouse emulator skills and solution!

## Installation

### Linux

Simply paste this command in your terminal:

```bash
sudo curl -L <correct_url> -o /usr/local/bin/keynav-game
sudo chmod +x /usr/local/bin/keynav-game
```
